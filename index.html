📧 Gmail Integration Guide

Your 30-day outreach sequence is built in. Here's how to connect it to Gmail — from zero-setup one-click drafts to a fully automated sequence that sends itself.

⚡ Level 0 — One-Click Gmail Draft (Built In, No Setup)

On any lead's script page, expand any email script and click 📧 Send. This opens Gmail in a new tab pre-filled with the script body, subject line, and lead's name/company already inserted. Hit Schedule Send to queue it for the right day.

✅ Already live — Leads tab → select lead → Scripts for This Stage → expand any email → click 📧 Send

Enable Gmail Schedule Send

Free Easiest

In Gmail compose window: click the ▾ arrow next to the blue Send button → "Schedule Send" → pick the right day and time. Batch-schedule all 8 emails in ~5 minutes when you add a new lead.

📋 Level 1 — Gmail Templates (Free, Saves Time Per Lead)

Save each of your 7 email scripts as a Gmail Template so you can load them instantly without copy-pasting from the CRM every time.

1
Turn on Gmail Templates Gmail → Settings (gear icon) → See all settings → Advanced tab → Templates → Enable → Save Changes.
2
Save each script as a template Open a new Gmail compose → paste the script body → click the three-dot menu (⋮) in the compose toolbar → Templates → Save draft as template → name it "Day 1 – Problem Email", "Day 3 – Bubble-Up", etc.
3
Use when adding a new lead Compose → ⋮ → Templates → Insert template → personalize the [Name]/[Company] fields → Schedule Send to the right day. Repeat for all 8 emails.
💡 Pro tip: Use Gmail's "Schedule Send" presets — "Tomorrow morning" for Day 1 emails is usually the right cadence.

🔗 Level 2 — Gmail + Zapier Automation (Recommended for Scale)

Connect this CRM's lead flow to Gmail via Google Sheets + Zapier. When you add a lead to a Google Sheet, Zapier fires all 8 emails automatically at the right delays — replies stop the sequence.

1
Create a "Benepath Leads" Google Sheet Columns: First Name, Last Name, Email, Company, Industry, State, Lead Date. Every time you add a lead here, Zapier fires the sequence.
2
Sign up at zapier.com — free plan covers ~100 tasks/month That's roughly 12 full 8-email sequences per month on free tier. Starter plan ($20/mo) removes limits.
3
Build a multi-step Zap: Google Sheets → Gmail (x8 emails with delays) Trigger: New row in Google Sheets. Then chain 8 Gmail "Send Email" steps, each preceded by a Delay step. Use {{First Name}} and {{Company}} as merge tags in the email body.
4
Add a "Filter" step after each delay — stop if replied In Zapier, add a Filter step that checks Gmail for a reply from that contact. If a reply exists, the Zap halts. This prevents follow-ups landing after someone has already responded.
Zapier Multi-Step Sequence (Gmail): Trigger → Google Sheets: New Row ├─ Gmail: Send Email (Day 1 – Problem Email, 0 delay) ├─ Delay: 2 days → Filter: no reply? → Gmail: Send (Day 3 – Bubble-Up) ├─ Delay: 2 days → Filter: no reply? → Gmail: Send (Day 5 – Push Text note) ├─ Delay: 2 days → Filter: no reply? → Gmail: Send (Day 7 – Make It Real) ├─ Delay: 4 days → Filter: no reply? → Gmail: Send (Day 11 – 2nd Problem) ├─ Delay: 4 days → Filter: no reply? → Gmail: Send (Day 15 – Right Person?) ├─ Delay: 4 days → Filter: no reply? → Gmail: Send (Day 19 – Pushaway) └─ Delay: 4 days → Filter: no reply? → Gmail: Send (Day 23 – Final)
⚠️ Important: Day 3/7 emails should be replies to the original thread — in Zapier's Gmail action, set "In Reply To" using the Thread ID from Step 1 to keep them in the same Gmail thread.

⚙️ Level 3 — Google Apps Script (Free, Fully Automated, No Zapier)

If you want zero monthly cost and full control, Google Apps Script runs directly inside Google Sheets and sends Gmail on a schedule — no third-party tools required.

1
Open your Google Sheet → Extensions → Apps Script This opens a free JavaScript environment that has full access to Gmail and your Sheet. No coding experience required — use the template below.
2
Paste a send-sequence script and set a daily trigger The script reads each row, checks how many days since the lead date, and sends the matching email if it's due. Run it on a daily time-based trigger (Triggers → Add Trigger → Time-driven → Day timer).
// Google Apps Script — daily email sequence function sendSequence() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); const today = new Date(); for (let i = 1; i < data.length; i++) { const [firstName, , email, company, , , leadDate, lastSent] = data[i]; if (!email || !leadDate) continue; const daysSince = Math.floor((today - new Date(leadDate)) / 86400000); const SEQ = [{day:1,subj:`${firstName}'s Health Insurance`,body:`Hi ${firstName}...`}, {day:3,subj:'RE: ...',body:`Hey ${firstName}, just tried you...`}]; const step = SEQ.find(s => s.day === daysSince); if (step && lastSent !== step.day) { GmailApp.sendEmail(email, step.subj, step.body); sheet.getRange(i+1, 8).setValue(step.day); // mark sent } } }

💡 Gmail Deliverability Best Practices

✓
Keep Day 3, 7, 15, 23 emails as replies to existing threads — In Gmail this means hitting "Reply" and keeping the same subject prefix (RE:). Zapier and Apps Script both support "In Reply To" using the thread ID from the first email.
✓
Best Gmail send times: Tue–Thu, 8–10am or 3–5pm (prospect's local time) — Gmail's Schedule Send lets you pick a specific time. Avoid Monday mornings and Friday afternoons.
✓
Stop automation immediately when someone replies — In Zapier add a "Filter" step; in Apps Script check for reply threads before sending. Nothing kills a warm lead faster than an automated follow-up after they've already responded.
✓
Send from your real Gmail, not a bulk tool — For cold outreach at under 50/day, your personal Gmail (or Google Workspace nick@bridgewaterbenefits.com) has better deliverability than Mailchimp or mass-send platforms, which are flagged as marketing.
✓
Warm up a Google Workspace domain first — If you're on a new domain, send 10–20 real emails per day for 2 weeks before starting automated sequences. This builds your sender reputation with Google.
